## 1. 서론 (Introduction)

본 보고서는 2026년 5월 중순의 매크로 금융 위기 상황(호르무즈 해협 봉쇄 우려, 미 CPI 3.8% 서프라이즈로 인한 인플레이션 재가속 등)에서 동일한 프롬프트를 부여받은 4대 글로벌 LLM의 출력 데이터와 통계치를 정밀 검증한 마스터 보고서입니다. 

4개 모델 모두 매크로 진단(국면 판정)에서는 높은 수준의 합의를 이루었으나, 구체적인 자산 배분(Weighting)과 위험 자산 스크리닝(Execution) 단계에서는 극단적인 분산이 발생했습니다. 이 현상이 가지는 금융공학적 의미를 분석하고, 실전 투자를 위한 최종 합의체(Ensemble) 운용 전략을 도출합니다.

## 2. 매크로 진단의 통계적 유의성 (Macro Composite Consensus)

4개 모델의 경기 침체 확률(5-Factor Recession Composite) 산출 결과는 통계적으로 매우 유의미한 수렴성을 보여줍니다.

| 모델 | 5-Factor Recession Composite | Phase 판정 | 특징 |
| :--- | :---: | :---: | :--- |
| **Claude** | 62.25% | Phase 3 | 중간값. 5개 컴포넌트 균형 가중 |
| **Gemini** | 63.30% | Phase 3 | 최고값. 호르무즈 봉쇄 + CPI 가산 강하게 반영 |
| **ChatGPT** | 58.00% | Phase 3 | 최저값. Phase 2(Late-Cycle) 경계면에 근접 판정 |
| **DeepSeek** | 61.00% | Phase 3 | 중간값. 주식형 방어주 팩터에 몰입 |
| **통계 지표** | **평균: 61.14%** | **표준편차: 2.29%p** | **변동계수(CV): 0.037 (3.7%)** |

### [금융공학적 해석]
변동계수(CV)가 5% 미만(3.7%)이라는 점은 **매크로 텍스트 및 정량 시계열 지표 분석에서 LLM 간의 정보 비대칭이 사실상 해소되었음**을 의미합니다. Sahm Rule 경고 진입(0.42%p) 및 ISM PMI 수축기 지속이라는 데이터가 각 모델 내에 동일한 가중치 리스크로 인식되었습니다. 즉, **"LLM을 활용한 탑다운(Top-Down) 매크로 국면 판정은 고도의 신뢰성을 가진다"**고 결론지을 수 있습니다.
