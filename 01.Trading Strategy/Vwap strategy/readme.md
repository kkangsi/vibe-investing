# VWAP 컨플루언스 거래량 매매 전략 해설

거래량(Volume)을 중심에 두고 VWAP · Bollinger Bands · RSI · MACD · 이동평균선(EMA)을 결합한 통합 매매 전략에 대한 설명서. 전략의 작동 원리와 함께 장점 · 단점 · 리스크를 정리한다.

---

## 1. 개요

VWAP 컨플루언스 전략은 단일 지표의 신호를 믿지 않고, **서로 다른 직무를 가진 지표들이 한 지점에서 정렬(confluence)될 때만** 매매하는 방식이다. 핵심은 가격에서 파생된 보조지표(BB, RSI, MACD, EMA)가 아니라, 가격 외부의 정보인 **거래량을 최종 승인 게이트(approval gate)** 로 둔다는 점이다.

핵심 명제: *가격은 "무엇이 일어났는가"를 말하고, 거래량은 "그것을 믿어도 되는가"를 말한다.*

---

## 2. 작동 원리 — 지표별 직무 분담

다섯 지표를 "다섯 개의 독립 확인표"로 취급하면 신뢰 과잉이 생긴다. BB · RSI · MACD · EMA는 모두 같은 종가에서 파생되어 서로 강하게 상관(multicollinearity)되기 때문이다. 따라서 각 지표에 **단 하나의 직무**만 배정한다.

| 지표 | 직무 | 답하는 질문 |
|---|---|---|
| 이동평균선 (EMA, 예: 200) | 추세 레짐 필터 | 롱만 볼까, 숏만 볼까? |
| VWAP + σ밴드 | intraday 공정가치, 진입 위치, 과열 청산 | 지금 가격이 비싼가 싼가? |
| Bollinger Bands | 변동성 상태 (squeeze ↔ expansion) | 에너지 응축인가 분출인가? |
| 거래량 (RVOL / Z-score) | 신호 승인 게이트 | 이 움직임을 믿어도 되는가? |
| RSI | 모멘텀 극단 · 다이버전스 | 추세가 지쳤는가? |
| MACD 히스토그램 | 진입 타이밍 | 지금이 진입 시점인가? |

### VWAP이 중심인 이유

VWAP(Volume-Weighted Average Price)은 거래량으로 가중한 평균 체결가다. 기관 투자자가 대량 주문을 **체결 벤치마크(execution benchmark)** 로 삼기 때문에, VWAP은 시장의 "공정가치"이자 자석(magnet) 역할을 한다.

- 가격 > VWAP: 매수측이 평균적으로 이익 중 → 매수 우위
- 가격 < VWAP: 매도측 우위
- VWAP ± kσ 밴드: ±1σ는 단기 과열/눌림, ±2σ는 통계적 이탈(평균회귀 기준)

---

## 3. 진입 · 청산 흐름

### 롱 진입 (모든 조건 충족 시에만)

1. **레짐**: 가격이 추세 EMA 위 (상승장 → 롱만 탐색)
2. **방향**: VWAP 상승 중 (기관 평균단가가 우상향)
3. **위치**: 가격이 VWAP −1σ 밴드로 눌림 (싸게 사는 구간)
4. **변동성**: Bollinger Band가 squeeze 이후 확장 시작
5. **거래량 승인**: 반등 봉에서 RVOL ≥ 1.5 (또는 Z-score ≥ 1) — 최종 게이트
6. **타이밍**: 최근 수 봉 내 RSI 50 상향 돌파 또는 MACD 히스토그램 음→양 전환

> 핵심 규율: 1~4와 6이 모두 맞아도 **5번(거래량 승인)이 빠지면 진입하지 않는다.** 거래량 없는 신호는 모두 잠정 신호로만 취급한다.

### 청산

| 우선순위 | 조건 | 행동 |
|---|---|---|
| 1 | 가격이 ATR 기반 손절선 하회 | 전량 청산 |
| 2 | 가격이 VWAP 하향 이탈 + 거래량 동반 | 전량 청산 (추세 무효화) |
| 3 | 가격이 VWAP +2σ 도달 | 부분 익절 (통계적 과열) |
| 보조 | RSI 베어리시 다이버전스 출현 | 트레일링 강화 |

## 4. 장점

| 장점 | 설명 |
|---|---|
| 거래량 기반 신호 검증 | 가격 외부 정보(거래량)로 가짜 돌파(false breakout)를 걸러내 신호 품질이 높다. |
| 직무 분리로 중복 제거 | 각 지표에 고유 직무를 배정해 동일 정보를 중복 카운트하는 신뢰 과잉을 줄인다. |
| 기관 동선 추종 | VWAP은 기관의 실제 체결 기준선이라, 대형 자금의 평균단가 부근에서 진입/청산하는 구조적 이점이 있다. |
| 명확한 손절 기준 | ATR · VWAP 이탈이라는 객관적 청산 규칙이 있어 손실 통제가 체계적이다. |
| 고선택성 (높은 기대 적중) | 컨플루언스가 정렬될 때만 진입하므로 신호당 기대 신뢰도가 높다. |
| 다중 시장 적용 | 거래량 정규화(RVOL/Z-score)를 거치면 주식·가상자산 등 단위가 다른 시장에도 동일 규칙 적용 가능. |

---

## 5. 단점

| 단점 | 설명 |
|---|---|
| 낮은 매매 빈도 | 여러 조건이 동시에 정렬돼야 해 진입 기회가 드물다. 기회비용·지루함이 발생한다. |
| 파라미터 과다 | 5개 지표 × 각 기간/임계값 → 조합 폭발. 과최적화(overfitting) 위험이 크다. |
| 다중공선성 잔존 | 직무를 나눠도 BB·RSI·MACD는 같은 가격 파생이라 완전한 독립 신호는 아니다. |
| 후행성 | RSI · MACD · 이동평균은 본질적으로 지연 지표라, 급변 구간에서 진입이 늦을 수 있다. |
| VWAP의 시간 종속 | intraday VWAP은 장 마감에 리셋되므로 스윙·장기 매매엔 부적합(→ Anchored VWAP 필요). |
| 학습 곡선 | 6개 요소의 상호작용을 이해하고 규율을 지키기까지 숙련이 필요하다. |

## 6. 리스크

| 리스크 | 내용 | 완화 방법 |
|---|---|---|
| 과최적화 리스크 | in-sample에서 거의 항상 좋아 보임 → 실거래 성과 붕괴 | Out-of-sample · Walk-forward 검증, 파라미터 ±1스텝 민감도 테스트 |
| 거래량 데이터 오염 | 가상자산은 wash trading으로 거래량이 부풀려져 승인 게이트가 오작동 | 신뢰 거래소·aggregated 데이터 사용, 건당 거래량 극단치 필터 |
| 횡보장 손실 누적 | 추세 없는 구간에서 거짓 신호·잦은 손절로 비용 누수(whipsaw) | EMA 레짐 필터 강화, squeeze 구간 진입 회피 |
| 미래참조 편향 | 백테스트에서 종가 확정 전 정보를 쓰면 성과 과대평가 | 신호는 종가 확정 후, 다음 봉 진입 전제로 검증 |
| 유동성·슬리피지 | 저유동 자산에서 VWAP 밴드 진입가와 실체결가 괴리 | 거래대금 하한 설정, 슬리피지·수수료를 항상 백테스트에 반영 |
| 레짐 전환 리스크 | 추세장 가정이 깨지는 변곡점에서 큰 손실 가능 | ATR 손절 엄수, 포지션 사이징(자본 대비 1~2% 위험)으로 1회 손실 제한 |
| 규율 붕괴 (심리) | 기회가 드물어 조건 미충족 상태에서 임의 진입하는 휴먼 에러 | 진입 체크리스트 기계화, 자동화/알림으로 재량 개입 차단 |

## 7. 적합 / 부적합 환경

| 적합한 환경 | 부적합한 환경 |
|---|---|
| 뚜렷한 추세 + 주기적 눌림이 있는 시장 | 방향성 없는 장기 횡보장 |
| 거래량 데이터가 신뢰 가능한 자산 | 거래량이 조작·왜곡된 저유동 자산 |
| intraday ~ 단기 스윙 (Anchored VWAP 시 스윙) | 펀더멘털·이벤트 주도의 갭 빈발 자산 |
| 충분한 유동성(낮은 슬리피지) | 호가 스프레드가 큰 박리 시장 |

---

## 8. 요약

VWAP 컨플루언스 전략은 "거래량 승인 없이는 진입하지 않는다"는 단일 규율 위에, 각자 다른 질문에 답하는 지표들을 결합한 **고선택성 추세추종 + 평균회귀 청산** 체계다. 강점은 신호 품질과 기관 동선 추종에 있고, 약점은 낮은 빈도와 과최적화 취약성에 있다.

가장 중요한 원칙: **백테스트가 아무리 좋아도 Out-of-sample 검증과 거래비용 반영을 통과하지 못하면 채택하지 않는다.** 지표는 의사결정을 돕는 도구이지 미래를 알려주는 신탁이 아니다.
