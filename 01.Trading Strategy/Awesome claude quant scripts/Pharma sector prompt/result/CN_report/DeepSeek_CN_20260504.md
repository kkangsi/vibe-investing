# 制药行业12个月展望：多空组合策略

## 执行摘要
基于“产品线-收入可见性-专利悬崖”三轴模型，对S&P 500及NASDAQ-100中市值超500亿美元的美国上市制药公司进行全面筛选。

当前宏观背景：  
2026年制药板块正经历历史上最密集的专利悬崖周期之一，约 **1,764亿美元年销售额面临LOE风险**。  

同时：
- GLP-1/代谢赛道持续扩张  
- 肿瘤免疫进入第二代机制迭代  
- 基因治疗加速验证  

👉 行业分化加剧，多空双向机会显著
