# Quant Prompt — 거시 변수 × M7·AI인프라: 레짐 분류 + 모멘텀 시그널

> vibe-investing 레포에 바로 넣을 수 있는 재사용 프롬프트 템플릿.
> 핵심 철학: **상관이 없으면 "없다"고 정직하게 말한다.** 거시로 설명 안 되는 종목에 억지 시그널을 만들지 않는다.
> 시그널 엔진은 **거시 레짐 분류 + 모멘텀 결합** (회귀 적정가 버전은 부록 참조).

---

## PROMPT A — 레짐 분류 + 모멘텀 (메인)

```
당신은 정직한 퀀트 애널리스트다. 거시 변수로 시장 '레짐(국면)'을 분류하고,
각 종목의 레짐별 과거 성과와 모멘텀을 결합해 매수/매도 시그널을 낸다.
통계적으로 정당화될 때만 시그널을 내고, 아니면 그렇게 말한다.

[입력]
- 거시(일별): DXY, UST10Y(미10년물), CPI_YoY, ConsumerSentiment(미시간), WTI
- 종목(수정종가): M7 + AI인프라(반도체 AVGO,AMD,TSM,ASML,MU /
  전력·데이터센터·네트워크 VRT,ETN,SMCI,DLR,GEV)
- 기간: 2020-01-01 ~ 2026-04-30

[레짐 분류 — 2축 4국면]
- 축1 금리추세: UST10Y 60일 변화 -> RATE_UP / RATE_DN
- 축2 위험선호: z(-DXY 60d) + z(WTI 60d) + z(ConsumerSentiment 60d 차분), >0이면 RISK_ON
- 4국면: Goldilocks(금리↓+선호), Reflation(금리↑+선호),
         Deflation/Recession(금리↓+회피), Stagflation/Shock(금리↑+회피)

[필수 검증]
1. 레짐별 종목 연율수익(레짐 에지)을 계산하고, Kruskal-Wallis로
   "레짐이 수익률을 유의하게 가르는가"(p<0.05)를 종목별로 검정하라.
   유의한 종목이 적으면 "레짐 효과는 방향성은 있으나 통계적으로 약하다"고 명시.
2. 시그널: 현재 레짐 에지 부호 + 50일 모멘텀 부호.
   둘 다 + : BUY / 둘 다 - : SELL/AVOID / 엇갈림 : HOLD.
   레짐 샘플<10이면 모멘텀만 사용.
3. 백테스트(룩어헤드 방지, 시그널 다음날 진입): CAGR/Sharpe/MDD를 B&H와 비교.
   **반드시 구분해 보고하라**: 이 전략이 (a)수익을 늘리는가 (b)낙폭을 줄이는가.
   레짐+모멘텀은 보통 상승장 수익은 일부 포기하지만 MDD를 크게 줄이는
   '리스크 관리형'이다. 이 성격을 정직하게 설명하라.

[출력]
- 레짐 분포 + 현재 레짐
- 레짐 에지 표(종목 × 4레짐 연율수익) + Kruskal-Wallis 유의성
- 현재 시그널 표: Stock | 현재가 | 레짐에지% | 모멘텀50d% | 시그널
- 백테스트 표 + "수익 vs 낙폭" 결론
거시·레짐이 종목을 설명 못 하면 억지 시그널 금지. 정직하게 말하라.
```

---

## 이번 분석에서 실증된 것 (합성 데이터, 방법론 검증용)

- **레짐 분류는 작동한다**: 2022 약세장이 Stagflation/Shock 국면과 정확히 겹쳤다.
- **레짐 에지는 방향성이 뚜렷**: 거의 모든 종목이 Stagflation/Shock에서 급락
  (TSLA -56%, MU -71%, META·DLR 음수), Goldilocks/Reflation에서 강세.
- **하지만 통계적으론 약함**: Kruskal-Wallis p<0.05는 17개 중 1개(MU)뿐.
  레짐 내 변동성이 커서 평균차이가 우연과 구별 어렵다. 표를 과신 금지.
- **백테스트 핵심**: Sharpe로 B&H 이긴 종목 5/17(상승장 수익 포기 비용).
  **그러나 MDD는 16/17 개선** (ASML -131%→-74%, SMCI -138%→-83%).
  => 이 전략의 가치는 **수익 극대화가 아니라 하락 방어**. 변동성 큰
     AI인프라(SMCI,MU,ASML)일수록 방어 효과가 크다.

## PROMPT C — 비지도 레짐 + 베타 중립 알파 (고급, 종목선택 검증용)

```
거시 변수에서 KMeans/HMM으로 레짐을 '비지도'로 추출하고(사람이 경계를
정하지 말 것), 종목 수익률을 시장지수(QQQ)로 회귀해 시장 부분을 제거한
'잔차 알파'에 레짐을 적용하라. 핵심 질문: 시장 전체가 아니라 '이 종목만의'
초과수익이 레짐에 따라 달라지는가?

[검증]
- KMeans와 HMM 결과를 Adjusted Rand Index로 비교(두 방법이 같은 구조를 찾나).
- HMM 상태 지속확률(전이행렬 대각)로 레짐이 끈끈한지 확인.
- 각 종목: total return의 레짐 의존(Kruskal-Wallis) vs residual alpha의 레짐 의존.
  잔차에서 유의 종목 수가 급감하면 -> 레짐 효과는 '시장 타이밍'이지
  '종목 선택'이 아니다. 그렇게 정직하게 결론지어라.
```

---

## ⭐ 이번 확장의 결정적 발견 (가장 중요)

- **비지도 레짐은 작동한다**: KMeans·HMM이 사람 라벨 없이 고인플레(CPI 8%+)
  국면을 독립적으로 분리. HMM 지속확률 0.997로 현실적으로 끈끈한 레짐. ARI 0.557.
- **시장 베타가 거대하다**: SMCI 1.81, NVDA·TSLA 1.64, MU·AMD 1.46.
  이 종목들은 시장의 1.5~1.8배로 움직인다 = 변동의 대부분이 '시장 공통'.
- **결정적 검정**: 거시 레짐 의존이 유의한 종목이
  **총수익 기준 3/17 -> 시장(QQQ) 제거 후 잔차알파 기준 1/17로 급감.**
- **결론**: 앞서 본 "Stagflation에서 다 무너진다"는 레짐 효과는 거의 전부
  **시장 전체가 함께 움직인 것**이다. 종목 고유 알파는 거시 레짐과 무관.
  => **거시 레짐은 "언제 시장에 있을지(market timing)"는 줄 수 있어도,
       "어떤 종목을 고를지(stock selection)"는 주지 못한다.**
  => 거시 기반 시그널은 포트폴리오 전체 익스포저 조절/하락 방어용으로 쓰고,
     개별 종목 알파는 펀더멘털·실적에서 찾아야 한다.

## PROMPT B — 거시 회귀 적정가 (부록, 보조 지표용)
거시를 연속 회귀로 적정가를 추정하는 방식. 단, 주간 R²가 1~3%에 불과해
(거시는 빅테크 주가 변동의 97%+ 설명 못 함) 보조 지표로만 쓸 것.
상세 로직은 `01_quant_engine.py` 참조.
